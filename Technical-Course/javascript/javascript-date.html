Date

The year, month, and day may be obtained using the JavaScript date object. A timer may be shown on a webpage with the use of a JavaScript date object.

You can create date objects using various date constructors. Provides methods to get and set the day, month, year, hour, minute, and second.

Creating Date Objects

There are four ways to create a date object.

  • new Date()
  • new Date(milliseconds)
  • new Date(Date string)
  • new Date(year, month, day, hours, minutes, seconds, milliseconds)
