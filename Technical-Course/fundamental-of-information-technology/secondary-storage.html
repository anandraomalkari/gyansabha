Secondary Storage

An addition memory called as secondary memory or auxiliary storage is used with most computer system. The secondary memory is non-volatile and has lower cost per bit.

The operating speed of secondary memory is slower the primary memory. This memory is used to store large amount of data permanently and the data can be transfer to primary memory whenever required.


They are two types of data
  1. Sequential Memory:

    In this device the data is accessed by sequencing through other location. It is suitable for application such as preparation of monthly salary, Monthly electricity bill, etc. In this method the records need to be processed on after another. Will working with a sequential access only by searching through all the address stored before it.

    They are one type of sequential memory:


    1. Magnetic Tape:

      It is the most popular storage medium for large data that are accessed and proceeded sequentially. Magnetic tape is a plastic required usually ½, ¼ inch. Wide 50 to 2400 feet long. it is clotted with are recording material that can be magnetised such as iron oxide, chromium dioxide the date recorded on the tape in form of tiny invisible magnetised and magnetised spot representing 1’s and 0’s on the clotted surface.

      A magnetic tape derived is used for storage and re-travel of data on the tape. The tape drive has read or write heads.

      Advantages:

      1. Storage capacity is very large.
      2. The cost per bit is very low.
      3. It is compact and light weight.
      4. They are easily portable.

      Disadvantages:

      1. As they are sequential they are not suitable random access of data.
      2. They must be stored in dust free environment.
  2. Direct Access

    In this device any data can be accessed immediately. They are most used in application where the data need to be accessed quickly. In banking system it may be required to determine the balance of a particular customer. If sequential access device is used then time taken to access this information will be very high. This can be avoided by using direct access device.

    They are two type of Direct Access Memory

    1. Magnetic Disk:

      It is a thin circular plate of metal or plastic and coated on both sides with a recording material such as iron oxide. The data is recorded on disk in from of magnetic and non-magnetic spots representing 0’s and 1’s.


      Storage Organisation:

      The surface of disk is divided into a number of constrains circle called as tracks.

      In addition to the circle, the disk surface is also divide into pie-shaped segment. If there are 8 segment then is track will be divided into 8 part each of these part of segment of track is called as sector.


      A sector typically contains 512 bytes to access data store on disk the recorded disk address must be specified. The disk address consists of sector number, track number, surface number.


      Types of Magnetic disk

      All the magnetic disks are round plotter they come in different sizes and use different by types of package there are 2 types of magnetic disk below:


      1. Floppy Disk:

        A floppy disk is a flat circular piece of flexible plastic clotted with magnetic oxide in cased in a square plastic cover. Floppy disk are called as floppy because are need of flexible plastic plates they were intrusted by IBM in 1972.

        A floppy disk drive is device used to read or write data from floppy disk. The speed of floppy is 300 to 400 rpm. 3.5 inch floppy disk are must commonly use floppy disk.


      2. Hard Disk:

        Round, flat piece of rigid metal (frequently aluminium) disks coated with magnetic oxide. Come in many sizes, ranging from 1 to 14-inch diameter.

        They are divided into 3 types:

        • Zip/Bernoulli Disks:

          It consists a single hard disk pattern stored in a plastic cartridge. The disk is commonly of 3.5 inch with storage capacity of 100MB. Its disk drive called as Zip drive can be portable or fixed.

        • Disk Pack:

          It consists of two or more hard disk platters mounted together its disk drive has a sprat read write had for each disk surface. Its disk drive is off interchangeable type and allows loading and unloading of different disk pack. Hence the disk pack has virtually unlimited storage capacity.

        • Winchester Disk:

          This disk consists of two or more hard disk platters. However unlike the disk pack drive, a Winchester disk drive is of fixed type. Hence disk have limited capacity. In this disk all the disk platter including the upper and lower surface are used data recording.


    2. Optical Disk

      An optical disk system consists of a rotating disk coated with then metal or some other material that is highly reflective. It uses laser beam technology for reading or recording of data on the disk surface.

      An optical dis has one long trick starting at outer edge and spiralling inward to the centre


      Types of Optical Disk:

      All optical disk are round plotters. They come in different size and capacity.

      1. CD ROM:

        CD ROM disk is a shiny, silver color disk usually of 5.25 inches. It is made of plastic and a layer of aluminium is applied to make the surface reflective. It is a read only storage medium and comes pre-recorded with information which can’t be changed. Its storage capacity is 650 to 700MB.

      2. CD-R/WORM:

        WORM disk allows the user to create they are own CD-ROM disk by using a CD-R drive. WORM disk loop same has standard CD-ROM disk. The data can be return only once on a WORM disk but can be read many times. Hence, like CD-ROM disk once data has been return on WORM disk it becomes permanent and can never be changed. These disk are also called as multiple section disk.

      3. CD-RW (Read/Write):

        CD-RW disk is very similar to a WORM disk with the change that we can less the previous contain and write on it multiple time. It is done by the leaser been with change the chemical properties during a writing. A CD-RW disk normal has a life time of 100 or more easer write cycles.

      4. DVD (Digital video/versatile Disk):

        DVD who’s designed most to store and distribute movies. It is similar to CD-ROM but it denser in recording data. Can be used for storage of large data. Allows storage of video in 4:3 or 16:9 aspect-ratios in MPEG-2 video format using NTSC or PAL resolution. Looks same as CD-ROM but has capacity of 4.7GB or 8.5GB.


    3. Memory Storage Device:
      1. Flash Memory:

        Flash Memory is a compact device of the size of pen and it comes in various shapes and designs. It enables easy transfer of data form one computer to another computer. A pen drive is a plug and play devices that plug into USB (Universal Serial Bus) port of a computer. The computer treated as removable drive. The user can read, write, copy, delete and move the data from computer hard disk to the pen drive or pen drive to hard disk. A pen drive doesn’t any better, cable and it is capability with computer having USB drive.

      2. Memory Card (SD/MMC):

        Similar to flash drive, flash memory card are available as removable storage drive in different electronic devices. Some of the popular memory card are SD/MMC. These cards can be used in virus digital device such as digital camera and mobile phone. Its helps in easy transfer of data from computer hard disk.
