System Software

Introduction

It is a set of one or more programmes designed to control the operation and existent the processing capability of a computer system. The following are function system software.

  • It supports development of their application software.
  • It supports execution of other application software.
  • It monitor effect use of various hardware resources such as CPU, Memory, I/O device, etc.
  • It communicate or control operation of peripheral devices such as monitor, printer, etc.

The programmes included in system software package are called system program and the programmers who prepare system software are known as system programmers. Some common types system programmes are:-

  • Operating System: OS software effectively and manage the utilization of all hardware software components of computer.

  • Programming language Translator: This translator transform the instruction prepared by the programmers in a programming language into a form that can be interpreted and executed by a computer system.

  • Communication Software: In a network environment the communication software enable transfer of data an programs from one computer system to another.

  • Utility Program: This program helps the users system maintains task such as taking back of formatting of hard disk, shorting of records, etc


Operating System

OS is an integrated set of programs that controls resources like CPU, memory, I/O device, etc. of a computer system and provide its user with an interface that is easy to use. The two objective of an operating system are:

  • Make a computer system easier to use:

    An OS hides the details of hardware resource form the programs and other user and provided them with a various interface for using a computer system it act as an intermediating between hardware and users as show in above drive hardware resource are surrounded by OS that is again surrounded by a liar of other system software (compiler, interrupter, utility software, etc.) and a set of application program. In the last lyre the users view the computer system in terms of user interface of application program.

  • Make a resource of a computer system:

    As operating system manages the resources by performing task such as keeping a track of which user is using what resource granting the resource request, maintaining the resource usage.


Utility Program

This program help the user in different system maintains task. Some of the common utility program are:

  • Disk Formatting:

    To use a new disk on a computer system. We need to format it according to the requirements of the computer OS. The disk formatting utility enable user to program.

  • Data Compression Utility:

    This utility program user a compression algorithm to transform file into a smaller size so that they required less storage space on the disk. The decompression algorithm is used to convert the compressed file into the normal form.

  • Data Backup Utility:

    The data stored in a computer can be cropped or losses because of different reasons. Hence, we most always keep extra copies (Backup data) of important data. A backup utility program enable creation of copy, pen-drive, magnetic disk, etc.

  • Anti-various Utility:

    A computer various is a program that does some damage to the computer system such as corrupting file or erasing the file. Anti-various utility identifiers types of various that has infected the computer by matching it with various definition.

    A good anti-various utility can clean different types of various normally and can be improved regularly by frequently updating it new various are disk current.
