Input Device

  1. Keyboard Device

    These are the most commonly used devices. They allow the data entry into computer by processing a set of keys. Mounted on a keyboard connected to a computer. The most popular keyboard is 101 keys QWERTY keyboard.

    QWERTY:
    • It is named after the first six letters of the top row of buttons.
    • It is the most commonly used keyboard with computers in modern times.

      • They are another two types of keyboard

      1. AZERTY Keyboard:

        It is considered the standard French keyboard. Its name is derived from the first six letters that appear on the top left row of the keyboard.



      2. DVORAK Keyboard:

        This type of keyboard layout was developed to increase the typing speed.



  2. Point and draw devices (Pointing Devices):

    The graphical User Interface (GUI) provides a screen with graphical icons or means and allows user to make fast selections.

    1. Mouse:

      Mouse is the most popular point and draw device. it is small hand-held device that fits o users palm. Comfortably it was more than 1 button on the taps.

    2. Trackball:

      A Trackball is a pointing device which has a roller ball placed on top of the trackball to move the cursor on the screen the ball is rolled with hand. A trackball required less than a mouse for operation.

    3. Joysticks:

      A Joysticks is a pointing device that works similar to a trackball to make the moment of the roller ball easier it is placed is socket with a stick mounted on it. The stick can be moved forward or backward or left and right. Joystick is mostly used in video game, training simulators, industrial robots, etc.

    4. Electronic Pen:

      It is a pen based point and draw device. A user holds the pen in his hand and points with it directly on the screen from the displayed menu or icons. A user can also draw graphics directly on the screen with it.

    5. Touch Screen:

      It is the most simple and easier of all input device. A touch screen enable a user to select from available option by simply touching the required icons or menu item display on the screen using fingers.

      Touch screens are mostly used in information kiosks which is an unattained located at public placed that’s stores information and allows common person’s to access stored information as per their requirement.

  3. Data Scanning Devices:

    Data Scanning Device are input device used for direct data entry into the computer from source documents.

    The data scanning device have following Charest:

    • The Eliminate the need manual entry.
    • The automatic entry or data improve data accuracy.

      • Commonly used Data Scanning Device:

      1. Image Scanner:

        An Image scanner is an input device that translate paper documents into an electronic format that can be store a computer. The input document may be typed text, pictures, graphics, etc. The 2 common types of image scanner are:

        • Flat head Scanner: It is like copy machine consisting of box having a glass plat an its task. The document to be scanned is less upside down on the glass plate.
        • Hand head Scanner: This Scanner has the set of LEDs present in small case that can be hand head. TO scan a document the scanner dodder slowly over it from 1 and 2 other end.
      2. OCR Device (Optical Character Recognition):

        The image scanner have following limitation.

        • As the scanned document is stored as an image insert of text, It is not possible to perform any word process of the document.
        • The storage space required for storing the document as an image is more that required for storing the same document as text.

        In OCR, the scanner is equipped with character reader software (OCR Software) that convert the image of character to equivalent ASCII code.

      3. OMR (Optical Mark Reader):

        This scanner are capable of recognizing a pre-specified type of mark by pencil or pen.

      4. Bar Code Reader:

        The data coded in form of small lines are knows as Bar code. The Bar code represent alpha numeric data by a combination of adjacent vertical line by varying their width and the space between them. Bar code are particular used for union identification of all types of books, how hold items, postal packages, etc.

        A Bar code reader is device used for reading bar coded data. it use laser bin technology. The most widely used bar coding system is UPC (Universal Product Code) which is of 10 digits.

      5. MICR (Magnetic Ink Character Recognition):

        MICR is similar to OCR. It is used by banking industry for faster processing of large volume of cheques. Bank identification code (name, branch, etc.), account number and cheque number are pre-printed using characters from a special character set on all cheques.

        It consists of numerals 0 to 9 and four special characters. MICR is not adopted by other industries because it supports only 14 symbols.

  4. Digitizer:

    A digitizer is an input device used for converting pictures, maps, drawing into digital form for storage in computer.

  5. Electronic card Reader:

    Electronic cards are small plastic card having encoded data for the application for which their used. It is normal connected to a computer and used to read a data encoded on an electronic card and transfers it to the computer.

  6. Speech Recognition Devices:

    This device are input device that allow a person to input data to a computer system by speaking to it. This device make computer is to use. This system are wield used in large number of application.

  7. Vision Input System:

    A vision input system allows a computer to accept to input data by viewing object. A computer with vision input device consistence of digital camera.

    The vision input device are many use in fact by robots used quality control and assembly processes.
