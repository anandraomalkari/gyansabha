PHP if-else Statement

Set conditions that are true or false, and then execute based on whether the condition is true or not.

There are 4 types of conditional statement:

  1. if Statement
  2. if-else Statement
  3. if-elseif-else Statement
  4. nested if Statement

  1. if Statement

    2. if statement is a decision making statement. The general from of if statement is as follows.

    How to Works:
    • The expression is true, then the statement are executed.
    • The expression is false, then the statement are not executed.
    Syntax

    If there is only one statement to be executed with if statement. Curly brackets ({}) are not required. When multiple statements are to be executed, all the statements are enclosed within a pair of brackets.

  2. if-else Statement

    3. if-else statement is more general as compared to if statement. In if-else statement of the boolean expression return true then the statements in “if block” are executed but boolean expression not return true then the statements in “else block” are executed.

    Syntax
    How to Work
    • The “if statement” is true, then the statement are executed.
    • The “if statement” is false, then the else statement are executed.

  3. if-elseif-else Statement

    4. It is used when more number of condition are involved in the given problem. The expressions are evaluated from top to bottom. When expression is found to be true, the related statement are executed and remaining part is skipped. If none of the expression returns true, no action takes place.
